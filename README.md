# dsipy

This repository provides the `dsipy` tool that allows creating some resources for the decentralized social identity, see [dsi-spec](https://github.com/Desvelao/dsi-spec) for more information.

Based on https://nfraprado.net/post/vcard-rss-as-an-alternative-to-social-media.html

# Join to the network

1. Create a vCard with your info according to the [dsi-spec](https://github.com/Desvelao/dsi-spec)
2. Optionally, create a rss or atom file for your feeds.
3. Host the files in a public-accesible web:
    - vCard
    - feeds (rss and resources referenced by the states)
4. Share the link to your vCard to your connections
